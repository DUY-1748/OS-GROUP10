# HOW-TO: Setup and Run the Virtual Memory Simulator (Group 08)

Hello Lecturer/TA,

This guide will help you set up the environment and run our Virtual Memory Management Simulator (OS_VME_08). Our application is built using Python with the built-in `tkinter` library for the GUI, so it requires minimal setup.

## 1. Prerequisites
- You need to have **Python 3.x** installed on your machine.
- No external libraries (like `pip install ...`) are required because we strictly used Python's standard libraries (`tkinter`, `csv`, `os`, `time`, `random`).

## 2. How to Run the Application
**Step 1:** Clone our GitHub repository to your local machine or download the source code as a ZIP file and extract it.

**Step 2:** Open your terminal (Command Prompt, PowerShell, or macOS/Linux Terminal).

**Step 3:** Navigate (`cd`) into the `Code` folder where our source code is located.

**Step 4:** Run the main application file by executing the following command:
   ```bash
   python main.py
